📋 How to Use These Demo Sites

  1. Make sure the Attribute Reliability Tracker extension is loaded at chrome://extensions
  2. Click the 🎯 icon in the toolbar → click ▶ Start Run → name it (e.g. "Lumina Store Test")
  3. Open one of the demo sites above
  4. The extension will auto-refresh each page 4× — wait for the overlay to complete on each page
  5. Navigate through all routes (Home → Products → Cart → Checkout → Account)
  6. Click ⏹ Stop Run in the popup when done
  7. View the results — Reliable tab should show: id, data-testid, aria-label, name, type
  8. Click 📋 Copy JSON to export with per-page score breakdown
  9. Click any attribute row to copy its name to clipboard
